git init . -> cria um repo git na pasta atual
git status -> mostra o status do trabalho (de cada arquivo)
git add nome_arquivo -> adiciona arquivo para ser commitado
git commit -m "mensagem" -> faz commit do que estava para ser commitado
git log  -> mostra todo o histórico de commit

## FLUXO DE TRABALHO

1. Criar o diretório do projeto
2. Iniciar o git dentro do diretório (pasta) criado (git init .)
3. Editar ou criar arquivos
4. Verificar o estado (git status)
5. Adicionar os arquivos (git add <nome-arquivo>)
6. Verificar o estado (git status)
7. Fazer o commit (git commit -m "mensagem")
8. Verificar o estado (git status)
9. Volta para o passo 3.
